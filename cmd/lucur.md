## BRIEF DESCRIPTION

lucur - liblucurious command line tool

## SYNOPSIS
**lucur** [options]

**lucur** --pde VK_QUEUE_GRAPHICS_BIT 

## DESCRIPTION

The **lucur** command line tool is used to assist in the construction of single application vulkan kms compositors
