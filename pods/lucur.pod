
=head1 NAME

lucur - liblucurious command line tool

=head1 SYNOPSIS

=over

=item B<lucur> [options]

=item B<lucur> --pde VK_QUEUE_GRAPHICS_BIT,1

=back

=head1 DESCRIPTION

The B<lucur> command line tool is used to assist in the construction of wayland compositors

=head1 OPTIONS

=over

=item The options that apply to the B<lucur> command are:

=item

=item B<-l, --pgvl>

=over

=item Print global validation layers that are only available if the vulkan sdk is installed.

=back

=item

=item B<-i, --pie>

=over

=item Print instance extenstion list.
Instance extensions are used to setup a device.

=back

=item

=item	B<-d, --pde> <VkQueueFlagBits,VkPhysicalDeviceType>

=over

=item Print device extenstion list.
Device extensions control the behavor of a particular device.

=back

=item

=item B<-v, --version	>

=over

=item Print lucurious library version

=back

=item

=item B<-h, --help>

=over

=item Show this message

=back

=back
